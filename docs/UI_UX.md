# UI/UX Guidelines

## Design Principles

1. Clarity over density
2. Immediate visual feedback
3. Safe editing with autosave
4. Low-friction navigation between months
5. Accessible labels and controls

## Core Screens

- Login
- Dashboard
- Project overview (12 months)
- Month editor (full-page canvas + grid)
- Asset library
- Events
- Templates
- Admin

## Editor UX

- Full-page A4 workspace
- Layer management
- Contextual property panels
- Grid edit mode and decorate mode
- Export modal with progress tracking

## Error and Status UX

- Friendly toast notifications
- Actionable validation messages
- Clear loading and empty states
