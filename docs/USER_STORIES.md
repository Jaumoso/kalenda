# User Stories

## Actors

- U: Standard user
- A: Administrator
- SYS: System

## Epic 1 — Project Management

- As U, I can create, duplicate, and delete calendar projects
- As U, I can track project progress month by month

## Epic 2 — Visual Editing

- As U, I can design each page with images, text, stickers, and backgrounds
- As U, I can reorder layers and adjust visual effects

## Epic 3 — Calendar Grid

- As U, I can customize grid colors, borders, typography, and layout
- As U, I can customize specific day cells

## Epic 4 — Assets

- As U, I can upload and organize assets in folders
- As U, I can search and reuse assets across projects

## Epic 5 — Events and Holidays

- As U, I can manage recurring and one-off events
- As U, I can include regional holiday data

## Epic 6 — Templates

- As U, I can save reusable templates and apply them across months

## Epic 7 — Export

- As U, I can export print-ready PDF/PNG output
- As SYS, export jobs are tracked and recoverable

## Epic 8 — Administration

- As A, I can manage user accounts and roles
- As U, I can manage profile and credentials
